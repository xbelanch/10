Pas 1. Presentació

La literatura infantil traspassa les barreres de l’edat; criatures i adults ens sentim sovint atrets per les seves històries, per les il·lustracions que els donen vida i pel tractament de temes universals que hi fan els autors. Però, davant la diversitat de publicacions, cap on ens decantem un cop som a la llibreria?

Amb aquest curs obtindràs algunes claus per saber mirar llibres per a infants i joves, els criteris per triar-los i valorar-los i, per extensió, t’acostaràs a la lectura i al món del llibre.

Síl·labus

El llibre com a producte cultural: els paratextos i les versions i traduccions
Familiaritza’t amb els elements externs que ofereixen informació sobre les obres i amb els criteris per detectar bones versions i traduccions.
El text i la imatge
Aprofundeix i analitza les característiques del text i la imatge així com les diferents relacions que s’estableixen entre aquests dos elements.
Els elements constructius de les històries infantils
Coneix els principals aspectes que conformen una història de ficció i els criteris per valorar-los.
Criteris per a una tria diversa de llibres infantils
Descobreix la diversitat de productes a l’abast.
Les característiques dels llibres per als més petits
Coneix les característiques específiques dels llibres per als infants de 0 a 3 anys.
Criteris de selecció per als lectors preadolescents
Aplica algunes recomanacions de selecció per als lectors més grans.
Les diverses funcions de la literatura infantil i juvenil
Tingues en compte les particularitats dels llibres si tens propòsits concrets per a la tria.
